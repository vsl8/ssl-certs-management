Format Guide
  • .PEM — Base64 encoded, most common format
  • .DER — Binary encoded certificate
  • .CRT / .CER — Certificate file (PEM or DER)
  • .PFX / .P12 — PKCS#12, contains cert + private key
  • .P7B — PKCS#7, contains cert chain only
  • .KEY — Private key file
  • .JKS — Java KeyStore, used by Java applications
